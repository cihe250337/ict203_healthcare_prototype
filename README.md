# HarborCare

HarborCare is a three-page responsive website for a family and primary care clinic. It is built with HTML, CSS, and vanilla JavaScript.

## Pages
- Home
- Services
- Contact

## Features
- Responsive layout for mobile, tablet, and desktop
- Accessible navigation and form structure
- Services filter
- FAQ accordion
- Contact form validation and success modal

## Project structure
- `index.html`
- `services.html`
- `contact.html`
- `styles.css`
- `script.js`
- `assets/`

## Run locally
Open `index.html` in a browser.
